# KOOWEXA

Plataforma para crear tu presencia virtual de forma rápida y sencilla.

## Sobre el Servicio

Se ofrecen las herramientas necesarias para mostrar tu presencia de forma virtual. Esto puede traer grandes beneficios **para ti y tu negocio**, si te interesa.

### Sencillo
Fácil de usar, sin necesidad de experiencia técnica.

### Rápido
Optimizado para conexiones lentas. Ideal para usuarios en Cuba.

### Hospedaje
Servidor incluido durante 3 meses. Sin complicaciones.

### Variedad
Puedes elegir entre: tienda, servicios.
